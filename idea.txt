Fault Systems -> when there is some fault in the system, we can query and get responses.
based on the type of response, we will get feedback

Optimal Test Sequencing: This is a classic NP-hard problem. Research often focuses on heuristics (Ant Colony optimization) to solve it

Bayesian Belief Networks (BBNs): Use these to model the causal relationships between components. When a "test" fails, you use Bayes' Theorem to update the probability of each component being the culprit.

The D-Algorithm: Specifically used in digital circuit testing to find "stuck-at" faults by propagating values through logic gates—very similar to how Wordle letters propagate constraints.

AO* Search -> using A* search algorithm on AND-graphs and OR-garphs. In a Graph: If node A is an OR-node with children B and C, you succeed if you solve B OR C.
Rollout algorithms -> this is similar to RL algorithms
